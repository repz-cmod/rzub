Requirements:

- You should have Java Open JDK 8 installed.
- You should have maven installed

To run on windows:

1- Create a mysql database for this app and run repz_db.sql inside db you created.
2- Run make.cmd file. It builds project and puts the jar file in run directory. YOU DO NOT NEED TO MAKE PROJECT UNLESS CODE IS UPDATED.
3- Copy env/config.json to run directory if you haven't already.
4- Configure config.json in run directory to point to the valid databases
5- Call run.cmd to run project
6- Configure and copy all files under plugins folder to iw4admin plugins folder

After first run, you dont need to run first 4 steps anymore.
